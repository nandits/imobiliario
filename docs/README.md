# Documentation

This folder contains the architecture, technology decisions, and implementation roadmap for the single-agent real estate website.

## Files

- [`architecture.md`](./architecture.md) — System design, data model, folder structure, and user flows.
- [`tech-stack.md`](./tech-stack.md) — Framework, styling, data layer, hosting, and cost breakdown.
- [`implementation-plan.md`](./implementation-plan.md) — Phased implementation schedule, milestones, and risk notes.

## Quick Start

1. Read `architecture.md` to understand the high-level design.
2. Read `tech-stack.md` to understand the tools and libraries selected.
3. Read `implementation-plan.md` to see the step-by-step build schedule.
